# Contributing to Agent Skill Lifecycle Kit

Thanks for your interest in contributing.

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Add or improve:
   - Skill checks
   - Pitfall handling
   - Templates
   - CLI workflows
   - Documentation
4. Run local checks
5. Open a pull request

## What good contributions look like

- Clear problem statement
- Small, focused scope
- Practical before theoretical
- Backward compatible when possible
- Includes tests or validation when relevant

## Areas especially welcome

- Better Skill quality rules
- Reusable maintenance templates
- Agent-specific adapters (Hermes / OpenClaw / Codex)
- Improved CLI ergonomics
- Documentation for real maintainer workflows
